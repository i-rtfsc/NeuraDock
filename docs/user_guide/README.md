## 快速参考

| 任务 | 操作方法 |
|-----|---------|
| 添加账号 | 账号 → 添加账号 → 填写表单 |
| 手动签到 | 点击账号卡片上的签到按钮 |
| 批量签到 | 选择账号 → 批量签到 |
| 启用自动签到 | 编辑账号 → 开启自动签到 |
| 刷新余额 | 点击账号卡片上的刷新图标 |
| 导出账号 | 选择账号 → 导出 |
| 导入账号 | 账号 → 导入 JSON |

## 键盘快捷键

| 快捷键 | 操作 |
|-------|------|
| `Cmd/Ctrl + N` | 添加新账号 |
| `Cmd/Ctrl + R` | 刷新所有余额 |
| `Cmd/Ctrl + ,` | 打开设置 |
